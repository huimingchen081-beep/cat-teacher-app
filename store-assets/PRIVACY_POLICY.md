# 猫咪老师 隐私政策

**最后更新日期：2026年6月16日**

## 1. 信息收集

猫咪老师尊重您的隐私。我们仅收集以下必要信息：

- **设备信息**：设备型号、操作系统版本（用于兼容性适配）
- **使用数据**：学习记录（年级、学科选择）、对话历史（存储在本地设备）

## 2. 权限使用说明

| 权限 | 用途 |
|------|------|
| 相机 (CAMERA) | 拍照扫描题目，用于AI解题功能 |
| 麦克风 (RECORD_AUDIO) | 语音输入提问功能 |
| 存储 (STORAGE) | 缓存学习数据、保存历史记录 |
| 网络 (INTERNET) | AI对话、图片识别、语音识别需要联网 |

## 3. 数据存储

- 学习记录和对话历史默认存储在设备本地
- AI问答请求通过加密连接发送至阿里云DashScope API处理
- 图片和语音数据仅用于当次请求，不持久存储

## 4. 第三方服务

本应用使用以下第三方服务：
- **阿里云 DashScope API**：提供AI对话、图像识别、语音识别能力
- 第三方服务的数据处理遵循各自隐私政策

## 5. 儿童隐私

本应用面向K-12学生。我们不会有意收集13岁以下儿童的个人信息。如发现误收集，请立即联系我们删除。

# Cat Teacher Privacy Policy

**Last Updated: June 16, 2026**

## 1. Information Collection

Cat Teacher respects your privacy. We only collect:

- **Device Info**: Model, OS version (for compatibility)
- **Usage Data**: Learning records (grade/subject selection), chat history (stored locally)

## 2. Permissions

| Permission | Usage |
|------------|-------|
| Camera | Photo scan for problem solving |
| Microphone | Voice input for questions |
| Storage | Cache learning data |
| Internet | AI chat, image/voice recognition |

## 3. Data Storage

- Learning records stored locally on device
- AI requests sent encrypted to Alibaba Cloud DashScope API
- Images/audio data used for current request only, not persisted

## 4. Third-Party Services

- **Alibaba Cloud DashScope API**: Provides AI chat, image recognition, speech recognition

## 5. Children's Privacy

This app is designed for K-12 students. We do not knowingly collect personal information from children under 13.
